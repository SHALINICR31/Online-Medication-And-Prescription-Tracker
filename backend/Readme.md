# Backend

This directory contains the backend implementation of **MEDLINK: Online Medication and Prescription Tracker**.

The backend is developed using **Spring Boot (Java)** and provides **RESTful web services** for handling application logic, authentication, and data management.

---

## Technology Stack
- Spring Boot (Java)
- RESTful Web Services
- MongoDB
- Spring Security with JWT

---

## Responsibilities
- User authentication and authorization  
- Prescription and medication data management  
- Medication intake tracking and reminders  
- Secure API communication with the frontend  

---

## Database
MongoDB is used to store user data, prescriptions, medication schedules, and intake records in a flexible and scalable format.
